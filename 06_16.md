# 자료구조

 - Stack : 선입후출 (ex. Main메서드 밑에 System.out.printf(%d\n,1))
 
# index
 - 검색속도를 높이기 위한 기술
 
# Spring DI
 - Dependency injection으로 관계 있는 것들끼리 자동결합을 시켜주는 것
 
# Spring AOP
 - Aspect Oriented Programming (ex. logging)
